-'ol' is used to create an ordered (numbered) list in HTML. 'ul' is used to create an unordered (bulleted) list in HTML. 'li' is used to define individual items within a list in HTML.

What is td and th in html? Write your answer here.

-'td' stands for "table data" and is used to define a standard data cell within an HTML table. 'th' stands for "table header" and is used to define a header cell within an HTML table, typically used for labeling or describing data in a column or row.
